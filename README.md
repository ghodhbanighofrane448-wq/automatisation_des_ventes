# Automatisation et Analyse des Ventes avec Python (Pandas & Matplotlib)

##  Description du projet
Ce projet consiste à automatiser l’analyse des ventes d’une entreprise e-commerce à partir d’un fichier CSV.
Le but est de remplacer un traitement manuel par un script Python capable de calculer automatiquement les indicateurs de performance.

---

##  Objectifs
- Générer un fichier de ventes (`ventes.csv`)
- Calculer le Chiffre d’Affaires Brut
- Appliquer les remises pour obtenir le CA Net
- Calculer la TVA (20%)
- Identifier le produit le plus rentable
- Exporter les résultats dans un nouveau fichier CSV
- Visualiser les résultats avec un graphique

---

##  Technologies utilisées
- Python
- Pandas
- Matplotlib
- VS Code
- Git / GitHub

---

##  Structure du projet
- `ventes.py` → génération et traitement de données statiques
- `ventes_dynamique.py` → lecture dynamique de fichiers CSV
- `main.py` → programme principal
- `ventes.csv` → fichier de données initial
- `resultats_final.csv` → résultats calculés
